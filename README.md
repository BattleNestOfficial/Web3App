# Neon Console PWA

Production-ready React + Vite PWA starter with:

- Responsive sidebar + top profile header
- Futuristic dark Tailwind UI
- Framer Motion page animations
- Installable PWA with offline caching

## Setup

1. Install dependencies:

```bash
npm install
```

## Run

```bash
npm run dev
```

Build production:

```bash
npm run build
npm run preview
```
